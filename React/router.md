## 3 Types of Components in React Router
1. Router Components
   - `<BrowserRouter>`
   - `<HashRouter>`
2. Route Matching Components
   - `<Route>`
   - `<Switch>`
3. Navigation Components
   - `<Link to='url'>Text</Link>`
   - `<NavLink to='url' activeClassName='class'>Text</NavLink>`
   - `<Redirect to='url'/>`
